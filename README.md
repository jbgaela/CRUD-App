
<h3>Config .env variables</h2>

Use the `.env.example` as reference to create your configuration file `.env`

The .env file should be placed at the root of /Backend folder.

```yaml
PORT=3000
NODE_ENV=development
DB_HOST=localhost
DB_USER=devuser
DB_PASSWORD=devpass
DB_NAME=devdb
```
